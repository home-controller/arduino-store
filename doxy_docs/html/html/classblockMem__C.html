Detailed Description

Wrapper for storageBlock_C but working with only 1 block id(eg all the 1wire chips saved info).

This works with a block of storage consisting of getSlots() slots all of the same size. e.g if we need to store 15 arrays of bytes each 7 bytes long. i.e. Equivelent to: byte arrayOfType[getSlots()=15][7];
