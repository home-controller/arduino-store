\doxysubsection{Detailed Description}
Wrapper for \doxyref{storage\+Block\+\_\+C}{p.}{classstorageBlock__C} but working with only 1 block id(eg all the 1wire chips saved info). 

This works with a block of storage consisting of \doxyref{get\+Slots()}{p.}{classblockMem__C_ab00c3bdc52c49a26f8b6931f9af058ab} slots all of the same size. e.\+g if we need to store 15 arrays of bytes each 7 bytes long. i.\+e. Equivelent to\+: byte array\+Of\+Type[\doxyref{get\+Slots()}{p.}{classblockMem__C_ab00c3bdc52c49a26f8b6931f9af058ab}=15][7];
